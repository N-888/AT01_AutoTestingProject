# Автоматическое тестирование с unittest в Python

Этот проект демонстрирует подход к автоматическому тестированию с использованием встроенного модуля Python `unittest`. Он включает набор математических функций и полный набор тестов, покрывающих основные сценарии, включая обработку исключений.

## ✨ Основные возможности

- **Модульное тестирование** – каждая функция проверяется изолированно.
- **Проверка равенства и неравенства** – использование `assertEqual` и `assertNotEqual`.
- **Проверка истинности/ложности** – демонстрация `assertTrue` и `assertFalse`.
- **Тестирование исключений** – проверка, что код корректно выбрасывает `ValueError` при делении на ноль.
- **Полное покрытие кода** – протестированы все математические операции: сложение, вычитание, умножение, деление, остаток от деления.
- **Структурированность** – тесты разбиты на классы по логическим группам, что упрощает поддержку.

## 🖼️ Визуальные подсказки (для PyCharm)

- **Создание файла**:  
  Нажмите правой кнопкой мыши на папке проекта → **New** → **Python File** → введите имя → Enter.
- **Запуск теста**:  
  Откройте `test.py`, нажмите зелёный треугольник в верхней панели (или `Shift+F10`).
- **Просмотр ошибок**:  
  Внизу откроется окно **Run**. Нажмите на красную строчку с ошибкой – увидите подробный стек и разницу между ожидаемым и реальным результатом.
- **Исправление кода**:  
  Перейдите в `main.py`, измените ошибочные утверждения в тестах на верные (например, `self.assertEqual(add(3, 7), 10)`) и перезапустите тесты – теперь все должны быть зелёными.
